Tag helpers represent the way to create custom html elements in asp.net core's Razor templating engine they replace the old and deprecated Html Helper facility.

This document is a cheat sheet that outlines briefly the key-points of using existing tag helpers will not be very helpful if you need to understand tag helpers from scratch

A lot of introductory resources are available online if you are not familiar with the concept. The following articles are a good starting point:

  • Introduction to Tag Helpers
  • Authoring Tag Helpers

Keypoints

  • Tag Helpers substitute markup in cshtml files with standard html markup:
    • {% highlight html linenos %} {% endhighlight %}

    turns to:

    {% highlight html linenos %}

    Hello World!

    {% endhighlight %}
  • Built-in tag helpers are available under the 
    Microsoft.AspNetCore.Mvc.TagHelpers
    assembly
  • Substitution can be activated by custom html tag name, custom html attribute and standard html elements name:
    • {% highlight html linenos %} {% endhighlight %}

Usage

  • Tag helpers need to be included in cshtml files by using the 
    @addTagHelper
    directive:
    • {% highlight cshtml linenos %} @* , *@ @addTagHelper HelloWorld, TechDominator.TagHelpersAssembly @*Possible to specify wildcard to include all tag helpers*@ @addTagHelper *, TechDominator.TagHelpersAssembly{% endhighlight %}
  • The default asp.net core project template includes 
    @addTagHelper
    in 
    View/Shared/_ViewImports.cshtml
    :
    • {% highlight c# linenos %} @using AspNetTagHelpersCheatSheet @addTagHelper *, Microsoft.AspNetCore.Mvc.TagHelpers @*usually custom tag helpers are included here*@ @addTagHelper *, TechDominator.TagHelpersAssembly{% endhighlight %}
  • There is a variety of useful built-in tag helpers (cache, img, ...)
  • Once included tag helpers can be used in cshtml files and IntelliSense support is available in visual studio
  • Some tag helpers provide custom attributes:
    • {% highlight html linenos %} {% endhighlight %}
  • Some tag helpers can accept a C# expression:
    • {% highlight html linenos %} {% endhighlight %}
  • It is possible to use standard html attributes:
    • {% highlight html linenos %} {% endhighlight %}

    turns to:

    {% highlight html linenos %}

    Hello World!

    {% endhighlight %}
  • It is possible to remove (uninclude) tag helpers from a specific view by using the 
    @removeTagHelper
    directive:
    • {% highlight html linenos %} @removeTagHelper img, TechDominator.TagHelpersAssembly{% endhighlight %}
  • It is possible to disable tag helper activation for specific element by using the 
    !
    operator:
    • {% highlight html linenos %} {% endhighlight %}

    turns to:

    {% highlight html linenos %} {% endhighlight %}
  • Some tag helpers can accept content and even nested elements which can be tag helpers as well
    • {% highlight html linenos %}

    Title

    {% endhighlight %}
